# drive'n'meet

Konzept-Showcase für ein Schulprojekt: eine App-Idee für junge fahrer mit Blitzer-Karte, Kennzeichen-Profil, Standort-Matching ("Convoy-Ping") und einem Community-Feed für geteilte Spots.

Alle Screens sind statische Mockups mit simulierten Daten — keine echte Standortverfolgung, kein echtes Backend, kein echtes Matching.
